## Тема
> Решение задачи мультилейбл-классификации на основе датасета CMU-MOSEI
## Задание
1. введение;
2. аналитический обзор существующих исследований;
3. эксполаторный анализ корпуса;
4. baseline. выбор 2 модальностей, извлечение различных признаков (min 2) для каждой модальности, построение нейронной сети прямого распространения для каждой модальности отдельно;
5. усложнение моделей. нейронные сети (CNN, Transformers, RNN..), подбор гиперпараметров;
6. объединение модальностей;
6. заключение.
---
## Цель
> разработка алгоритма мультилейбл-классификации для распознавания шести базовых типов эмоций на основе корпуса CMU-MOSEI для повышения количественной оценки качества работы моделей через метрики F1-score и Accuracy для каждой модальности
## Задачи
1. провести аналитический обзор существующих архитектур нейронных сетей и методов слияния признаков, применяемых в задачах мультимодального распознавания эмоций;
2. выполнить предобработку данных корпуса CMU-MOSEI: извлечение признаков из текстового и акустического каналов, и преобразовать значения интенсивности в бинарные метки для мультилейбл-классификации;
3. реализовать программную модель мультимодальной классификации, способную одновременно обрабатывать и интегрировать разнородные входные сигналы;
4. провести экспериментальную оценку работы модели на тестовой выборке и проанализировать влияние каждой модальности на итоговую точность распознавания эмоциональных состояний.
